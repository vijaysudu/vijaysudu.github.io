Part 2: Data Visualization¶

Generally, you want the fastest Pokemon with the strongest attack and strongest defense for it to be considered well rounded. Here is a quick explanation of all the various columns:

: ID for each Pokemon¶

Name: Name of each Pokemon

Type 1: Each Pokemon has a type, this determines weakness/resistance to attacks

Type 2: 2nd type of Pokemon (will not be used in this tutorial!)

Total: sum of all stats which gives a general idea of a Pokemon's overall strength

HP: hit points, or health, defines how much damage a Pokemon can withstand before fainting

Attack: the base modifier for normal attacks (eg. Scratch, Punch)

Defense: the base damage resistance against normal attacks

SP Atk: special attack, the base modifier for special attacks (e.g. fire blast, bubble beam)

SP Def: the base damage resistance against special attacks

Speed: determines which Pokemon attacks first each round
